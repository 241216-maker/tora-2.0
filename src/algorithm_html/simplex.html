Criterios Matemáticos

El Método Simplex evalúa los puntos extremos algebraicos en las fronteras del área de solución para optimizar recursos eficientemente.

CRITERIO 01

Condición de Entrada: Entra a la base la variable con el coeficiente neto más negativo en la fila $Z$ (para Maximización).

CRITERIO 02

Condición de Salida: Abandona la base la fila con la menor razón aritmética estrictamente positiva ($\text{RHS} \div \text{Pivote}$).

CRITERIO 03

Holguras Activas: El algoritmo introduce variables de holgura para transformar desigualdades de tipo ($\le$) en ecuaciones puras.
